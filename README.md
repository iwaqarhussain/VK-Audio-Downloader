# VK-Audio-Downloader

An utility that downloads music from VK community/user page.

## Requirements:

- `Node 10+`

- `Python 3.7`

- `FFMpeg in PATH`

## How-to:

1. `npm ci`

2. `npm start`

3. Navigate to audio page

4. Wait... Just wait...

5. `pip3 install -r Music/requirements.txt`

6. `python3 Music/m3uNormalizer.py`
